Web Scraper Features

  • Customizable Search The scraper takes user input for search keywords and generates a URL, ensuring it can be adapted for various queries on Al Jazeera.
  • Handles Multi-word Keywords Automatically encodes spaces as %20 for multi-word search queries, ensuring seamless URL formation like "tariffs%20china".
  • "Show More" Handling The scraper intelligently detects and clicks the "Show More" button to load additional articles, continuing to scrape as long as new content is available.
  • Article Scraping For each article, the scraper extracts the title, content, article URL, and the original search keyword for precise documentation.
  • User-Agent Rotation To prevent being blocked, the scraper rotates user-agents with each article visit, mimicking human browsing patterns.
  • Random Delays To avoid rapid requests and reduce detection, the scraper waits randomly between 3-5 seconds between article visits.
  • CSV Output The scraper efficiently saves all scraped data (article title, content, URL, and search keyword) into a CSV file, appending new data or creating the file if it doesn't exist.
  • Headless Browser The scraper uses Selenium with a headless browser for executing dynamic JavaScript and loading full content from the search results page.
  • Error Handling Logical print statements guide the user through the scraping process, providing feedback on each action and any issues encountered.
